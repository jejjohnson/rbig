# RBIG Package

This package implements Rotation-Based Iterative Gaussianization (RBIG).

## Module Structure

- `_src/base.py` - Base RBIG class
- `_src/model.py` - AnnealedRBIG model
- `_src/mi.py` - Mutual information via RBIG
- `_src/kld.py` - KL divergence via RBIG
- `_src/metrics.py` - Information-theoretic metrics
- `_src/jacobian.py` - Jacobian computation
- `_src/marginal.py` - Marginal Gaussianization utilities
- `_src/rotation.py` - Rotation utilities
- `_src/densities.py` - Density estimation utilities
- `_src/parametric.py` - Parametric transformations
- `_src/image.py` - Image utilities
- `_src/xarray_st.py` - xarray compatibility
- `_src/xarray_image.py` - xarray image utilities
